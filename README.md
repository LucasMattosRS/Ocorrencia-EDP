# ⚡ EDP Ocorrência Automation

Automação de preenchimento do formulário "Nova Ocorrência" do SGS - EDP.

A pessoa preenche apenas **Descrição** e **Endereço** no celular, e o sistema gera um código que preenche automaticamente todos os campos fixos no SGS.
